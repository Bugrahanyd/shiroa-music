# SHIROA — AI-powered Music Production & Licensing

**SHIROA** is an AI-driven music production and exclusive-licensing platform by HydRaboN.
Goal: build a professional music licensing marketplace (Phase 1) and evolve into an AI music production & studio platform (Phase 2).

## Repo structure (plan)
- /docs              — project documentation (this folder)
- /frontend          — Next.js frontend (to be added)
- /backend           — Nest.js backend (to be added)
- /ai_service        — AI microservices (to be added)
- /infra             — Terraform / IaC (to be added)

## How to use
1. Read docs/TECHMAP.md and docs/ROADMAP.md to understand architecture & plan.
2. Clone repo and follow docs/DOMAIN_SETUP.md to prepare environment.
3. Add features in feature branches and open PRs to `develop`/`main`.
