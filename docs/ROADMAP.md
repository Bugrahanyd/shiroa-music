# ROADMAP — SHIROA (High-level)

## Phase 0 — Foundation
- Docs + repo baseline (done)
- Domain + Cloudflare + Vercel baseline
- Infra accounts (AWS/GCP), DB accounts (MongoDB Atlas, Postgres)

## Phase 1 — MVP
- Admin upload + transcode + preview streaming
- Track detail + buy flow (Stripe)
- Basic analytics & popular lists

## Phase 2 — Growth
- Credit system, studio v1
- Prod onboarding for producers (manual)
- Recommendation engine (simple collaborative / popularity based)

## Phase 3 — AI
- AI microservices: Composer, Vocalizer, Mixer
- Studio v2 (AI editing)
- Model training pipeline, labeling team

## Phase 4 — Scale & Enterprise
- SLA, global CDN tuning
- Licensing platform for enterprise customers
- Marketplace features (label dashboard, analytics for buyers)
